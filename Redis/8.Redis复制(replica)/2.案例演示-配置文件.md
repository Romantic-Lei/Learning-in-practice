# 案例演示-配置文件

### 架构说明

一个Master两个Slave，三台虚拟机，每台都安装redis

![](images/3.主从架构.jpg)

拷贝多个redis.conf文件

redis6379.conf、redis6380.conf、redis6381.conf

### 小口诀

三台虚拟机需要能相互ping通且需要注意防火墙配置

三大命令：

1. 主从复制

   replicaof 主库IP 主库端口，配从(库)不配主(库)

2. 改换门庭

   slaveof 新主库IP 新主库端口

3. 自立为王

   slaveof no one
